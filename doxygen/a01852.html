Detailed Description

and Bits

All registers are named ADXL345_name_REG where name is the name of the register as described in the sensors datasheet.

All bit positions are named ADXL345_name for single bit entries and ADXL345_name_L for multiple bit entries, where the position always means the entries LSB.
