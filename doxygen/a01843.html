Detailed Description

INGA is an Open Source Wireless Sensor Node for many different applications.

INGA was developed at IBR as Inexpensive Node for General Applications and became part of many projects.

Features

  • Transceiver
  • Temperature sensor
  • Gryoscope
  • Accelerometer
  • Pressure sensor
